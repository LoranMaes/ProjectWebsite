Puzzle: Nokia

Electric scheme

Explanation

Deze puzzel werd gemaakt door Jakob.
Nadat de code correct is ingevoerd met de potentiometers zal er een nieuw geluidje afspelen. Hierin is te horen dat het vliegtuig aan het crashen is en dat er een telefoon moet worden opgeladen. Vanaf de lader in de telefoon zit zal er een code aan de hand van ledjes achter de nummers van de nokia oplichten. Deze nummers vormen de nieuwe code die zal moeten ingevoerd worden op het numpad die dan leid naar de RFID scanner puzzel.
